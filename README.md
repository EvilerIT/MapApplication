# 说明

实现了打开三方地图导航功能，包括高德、百度、腾讯地图









| 名称 | 含义 | 必要性 |其他 |说明 |
| :----:|  :----: | :----: |:----: |:----: |
| mode | 交通方式 | 选填 |公交、驾车、骑行、步行 |默认为”骑行“ |
| latitude|	维度|	必填|		eg. 39.914935|		坐标系为BD09|	
|	longitude	|	经度|		必填	|	eg. 116.404269	|	坐标系为BD09|	
|	destination	|	目的地名称	|	选填	|	eg. 天安门|		默认为”目标地址“|
